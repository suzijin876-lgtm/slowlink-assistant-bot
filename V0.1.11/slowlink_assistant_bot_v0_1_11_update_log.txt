SlowLink Assistant Bot V0.1.11 更新日志

定位：丰富定时报表信息，并让服务器报表日志完全中文化。

改动：
1. 日报不再使用“约”，直接显示 Bot 实际转发数量。
2. 日报增加统计日期、活跃时段、首次转发和最后转发时间。
3. 周报和月报增加完整统计周期、日均转发、最活跃日期和最后转发时间。
4. 空报表统一显示转发 0 条、运行状态待命中和异常记录 0 次。
5. 当前概览移除“约”字样。
6. 服务器日志使用“日报、周报、月报”，不再显示 daily、weekly、monthly 等内部类型。
7. 报表发送完成日志同时记录统计周期、转发数量和异常数量。
8. 新增报表活动统计和中文日志回归测试，测试总数增加到 50 项。

验证：
- Python AST 语法检查
- python -B -m unittest discover -s tests -v

部署注意：
- 保留服务器 .env、data/ 和 SQLite 数据。
- 只重建 slowlink_assistant_bot，不影响主 SlowLink。

SlowLink Assistant Bot V0.1.10 更新日志

定位：稳定性和轻量性能优化，不改功能边界。

改动：
1. 主轮询循环在收到更新时不再额外等待 poll_interval，消息堆积时可以更快追上。
2. Telegram API 请求增加一次轻量重试，覆盖临时网络错误、429 限流和 5xx 服务端错误。
3. SQLite 转发记录增加报表查询索引，记录变多后统计更稳。
4. 读取 update offset 和报表发送状态时补充线程锁，减少并发读写边缘风险。
5. Docker 构建上下文排除版本归档、文档、部署脚本和 zip 包，服务器重建更轻。

验证：
- python -m compileall -q assistant_bot tests
- python -m unittest discover -s tests -v

部署注意：
- 保留服务器 .env、data/ 和 SQLite 数据。
- 只重建 slowlink_assistant_bot，不影响主 SlowLink。
